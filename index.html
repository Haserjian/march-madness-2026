How predictions are made: Consensus odds from every major US sportsbook, averaged after removing the house’s profit margin. No AI model — the market IS the prediction. Every day’s picks are digitally signed before tipoff. If any prediction is changed afterward, the signature breaks.
Digital signatures (Ed25519) — the same technology behind SSH and software updates. This is not betting advice.

Verify any day's predictions yourself (requires Python):

Checks that the digital signature is valid — proving the predictions existed before tipoff and haven’t been changed.

pip install pynacl
curl -s https://haserjian.github.io/march-madness-2026/attestations/2026-02-24.json | python3 -c "
import base64, json, sys
att = json.load(sys.stdin)
from nacl.signing import VerifyKey
vk = VerifyKey(base64.b64decode(att['signer_pubkey']))
unsigned = {k: v for k, v in att.items()
            if k not in ('signature', 'signer_pubkey', 'signer_pubkey_fingerprint')}
vk.verify(json.dumps(unsigned, sort_keys=True, separators=(',', ':')).encode(),
          base64.b64decode(att['signature']))
print('PASS:', att['date'], att['n_games'], 'games')
"
